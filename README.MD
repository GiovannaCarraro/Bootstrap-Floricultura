# Projeto FloraGrid

Este projeto e um site de floricultura feito com HTML, CSS e Bootstrap.

O que o site faz:
- Mostra uma vitrine de flores.
- Tem um contador de carrinho que aumenta ao clicar nos produtos.
- Tem um formulario de contato que mostra um aviso de enviado.
- O layout funciona tanto no computador quanto no celular.

Como usar:
1. Salve o codigo em um arquivo index.html
2. Abra o arquivo no seu navegador.
3. Precisa de internet para carregar o visual do Bootstrap.

Tecnologias:
- HTML
- Bootstrap (CSS e Icones)
- JavaScript (Logica do carrinho e alertas)
