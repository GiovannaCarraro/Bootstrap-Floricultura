# 🌸 FloraGrid | Floricultura

Projeto de uma landing page moderna para uma floricultura fictícia chamada **FloraGrid**, desenvolvida com HTML, CSS, JavaScript e Bootstrap.

---

## ✨ Funcionalidades

- Layout moderno e responsivo
- Navbar com menu dropdown
- Vitrine de produtos
- Galeria de imagens
- Sistema simples de carrinho
- Formulário de contato
- Alertas personalizados com SweetAlert2
- Efeitos hover e interações visuais

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Bootstrap Icons
- SweetAlert2

---

## 📁 Estrutura do Projeto

```bash
FloraGrid/
│
├── index.html
├── assets/
│   └── css/
│       └── style.css
```

---

## 🚀 Como Executar

1. Faça o download ou clone este repositório:

```bash
git clone https://github.com/seu-usuario/floragrid.git
```

2. Abra a pasta do projeto

3. Execute o arquivo:

```bash
index.html
```

Ou utilize a extensão **Live Server** no VS Code.

---

## 🌷 Recursos da Interface

### 🛒 Carrinho Dinâmico
Ao clicar em **Adicionar ao Carrinho**, o contador do carrinho é atualizado automaticamente.

### 📩 Formulário de Contato
O formulário possui validação básica e alerta visual de envio.

### 🌟 Alertas Interativos
Utilização do SweetAlert2 para mensagens elegantes e modernas.
